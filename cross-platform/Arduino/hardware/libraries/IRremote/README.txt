Esta version de IRremote esta modificada para poder usarla con el TIMER1 de 16 bits (ojo, que hay que cambiar el top 
value de 256 a 65536) de algun micro cualquiera.
Por ahora, para el IR send, están harcodeados los valores de los pines, y usa también el timer 1. 

Se probó con los sensores PNA4602M y TSOP34338.
